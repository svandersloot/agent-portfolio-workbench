# Prompt

Run the Runtime Context Map pilot through Agent Lab for Jira Work Item Assistant and Release Notes Manager.

Check whether the new map improves each agent's source routing before any manual Studio transfer:

- Does each pilot agent have a compact Runtime Context Map?
- Does each map point user/task intent to the smallest trusted source?
- Does each map name forbidden substitutes?
- Does each map define fallback behavior when the mapped source is missing?
- Does each map include smoke prompts tied to existing evaluation material?
- Do the Studio setup pages tell a human how to verify the map after manual setup?

Do not call live ROVO Studio, Jira, Confluence, or external systems.
