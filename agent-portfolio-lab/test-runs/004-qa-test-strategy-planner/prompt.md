# Prompt

Run the QA Test Strategy Planner candidate through Agent Lab before any Confluence publish or manual Studio configuration.

Check whether the local reusable package, sanitized fixtures, and Studio-style CSV cover:

- Strategy draft from supplied epic and partial QA context.
- Plan quality review against QA criteria.
- Readiness gap analysis.
- Automation-suite planning handoff.
- Routing to QA Test Case Architect v2, Jira Work Item Assistant, Release Drift Monitor, Release Health Analyst, and Performance Test Report Agent where appropriate.
- Guardrails for no Jira writes, no Confluence publish, no Studio write, no final QA approval, no go/no-go, and no unsupported test skipping.

Do not call live ROVO Studio, Jira, Confluence, test tools, or external systems.
