# Manual Promotion

Use this folder to prepare and track manual transfer of lab-proven instructions into Confluence, Rovo Studio, and manually configured automation workflows.

The lab must not claim a page is published, an agent is configured, or automation is live unless the user supplies that evidence.

## Promotion Evidence Labels

- `User-provided`: the user confirms publication or configuration.
- `Source-backed`: the user supplies a page export, Studio config excerpt, or workflow configuration excerpt.
- `Data Incomplete`: evidence is missing or partial.
- `UNKNOWN`: status cannot be determined from supplied context.

## Recommended Cycle

1. Update lab files.
2. Run local manual tests.
3. Prepare manual promotion report.
4. User manually updates Confluence, Rovo Studio, or workflow config.
5. User supplies confirmation or config excerpts.
6. Run Studio parity or workflow configuration review.
