# Sanitized Release Health Pages

Store sanitized release health page excerpts used for local-only portfolio tests.

Use placeholders for team-specific values, including `[release name]`, `[fixVersion]`, `[Jira project]`, `[release owner]`, `[QA lead]`, `[dev lead]`, and `[Confluence page]`.

Do not include live internal URLs, real page IDs, tokens, owner names, or production-only details.
