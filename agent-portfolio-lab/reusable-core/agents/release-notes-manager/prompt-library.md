# Release Notes Manager Prompt Library

## Draft Note Candidates

```text
Using only supplied release notes source material, draft concise release-note candidates for [release name]. Label evidence, list publication blockers, and ask questions for unsupported claims.
```

## Review Draft Notes

```text
Review these draft release notes for unsupported claims, missing audience, unclear note type, and readiness concerns. Return short action tables only.
```

## Notes to Health

```text
Identify any note candidate that raises a release readiness concern and convert it to the Notes to Health handoff contract.
```

## Drift to Notes

```text
Using this Health or Drift to Notes handoff, draft local note candidates. Do not publish or imply approval.
```
