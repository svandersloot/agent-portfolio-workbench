# Release Naming

Example release naming overlay.

| Item | Example Pattern |
| --- | --- |
| Release display name | `[release name]` |
| Fix version | `[fixVersion]` |
| Health page title | `[release name] Release Health` |
| Release notes title | `[release name] Release Notes` |

Agents must not invent release names. Use `UNKNOWN` when the name is not supplied.
