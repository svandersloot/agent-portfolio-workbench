# Governance and Safety

## Purpose

Use this section as the home for ROVO agent governance standards, context readiness guidance, measurement practices, risk controls, and builder-facing safety guidance.

This page should help builders and reviewers answer three questions:

- Is this workflow ready for agent design?
- Is the proposed solution governed at the right level?
- How will we measure whether the agent is useful and safe to keep?

## Core Pages

| Page | Purpose |
|---|---|

| Context Readiness Checklist | Helps reviewers determine whether source context is clear enough for durable agent behavior. |

## Operating Principles

- Prefer the least complex governed solution that solves the workflow.
- Reuse or extend existing agents before creating new ones.
- Scope knowledge sources narrowly and explicitly.
- Treat source cleanup as a prerequisite for reliable agent behavior.
- Require human approval for write-capable or high-risk actions.
- Use evaluations and measurement as evidence, not decoration.
- Preserve page history by moving pages instead of copying/recreating them when reorganizing the space.
